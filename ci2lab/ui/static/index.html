Uso de tokens

Contador de la conversacion

Entrada ultimo turno 0 Prompt, historial y contexto enviado al modelo.
Salida ultimo turno 0 Texto generado por el modelo.
Total ultimo turno 0 Entrada + salida.
Total conversacion 0 Suma acumulada de llamadas del chat actual.

Calculo

Total = tokens de entrada + tokens de salida. Los valores vienen del proveedor local cuando Ollama los devuelve.
